this is my config for tmux, neovim, and alacritty. clone this repo into `~/.config` 

```console
cd projects
nvim .
```

neovim automatically downloads and sets up the dependencies you need. you also need to run `:MasonInstallAll` to install the preconfigured lsps. 
you can use `:Mason` to find and install other lsps, linters and formatters you might need.
